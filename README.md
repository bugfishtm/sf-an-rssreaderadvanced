## 📙 Introduction 

RSS Reader Advanced is a clean, modern Android RSS/Atom feed reader built as a ready-to-use Android Studio project. It lets you subscribe to any number of feeds, organize them as sources, browse all articles in one place, and keep your favorite items for later. It ships with handy extras like OPML import/export, article search, mark-all-as-read, background refresh with notifications, predefined feeds to get started quickly, dark mode, and localization into many languages. The project builds from a single codebase into two product flavours — a free (Lite) build and an unlimited Pro build. All you need is Android Studio installed on your system, and you can quickly make the necessary changes.

## 🛠️ Installation

1. Method: Open the Suitefish Android application, browse the software store, and select the desired module. You can then effortlessly download and install it directly through the interface.

2. Method: Unzip the module’s ZIP archive and move the app.apk to your phone storage. Install by clicking in the Android File Manager.
